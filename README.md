# 🛒 Blinkit Sales Analysis Dashboard (Power BI)

## 📊 Project Overview

This project analyzes Blinkit's sales data to uncover insights about:

* Outlet performance
* Product categories
* Customer preferences
* Sales trends over time

## 🚀 Key Insights

* Total Sales: $1.2M
* Highest sales from Tier 3 outlets
* Fruits & Snacks are top-selling categories
* Supermarket Type1 contributes the most revenue

## 📈 Dashboard Features

* KPI Cards (Sales, Avg Sales, Ratings)
* Outlet Size & Location Analysis
* Item Type Distribution
* Fat Content Analysis
* Time-based Sales Trends

## 🛠 Tools Used

* Power BI
* Data Cleaning & Transformation
* DAX (Data Analysis Expressions)
